## üöÄ Getting Started

### Prerequisites
- Python 3.x installed on your system
- Basic understanding of programming concepts (optional)

### Running the Code

1. Navigate to the project directory:
   ```bash
   cd Basic_python_codes
   ```

2. Run any Python file:
   ```bash
   python filename.py
   ```

   Example:
   ```bash
   python 1_Types_of_variables.py
   ```
